HTML

  • HTML stands for HyperText Markup Language
  • HTML is the structural langauge of a webpage, composed of tags and elements
  • The head element contains information (metadata) about the webpage.
  • Metadata is seen by the browser and not by the user
  • Metadata gives the browser certain HTML instructions
  • The body element represents the visible content shown to the user.
  • The body element is known as a "parent element" and its child elements are the tags nested within it (h2, ul, etc.)
  • HTML is the starting point for a webpage, used in part with JavaScript and CSS, to create a completed website

CSS

  • CSS stands for Cascading Style Sheets
  • CSS tell the browser how to display the text and other content that you write in HTML
  • Provides a webpage with organization, aesthetic, and style
  • It uses properties
  • class and id selectors
  • box model

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Used for version control and debugging
  • GitHub
  • git add -A

JavaScript

  • Object-oriented programming language
  • Dynamic and high-level, JavaScript adds interactivity to a website
  • Uses variables, arrays, strings, numbers, and data types, among other things, to power a static HTML/CSS webpage
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The first step in learning complex programming
  • Case sensitive and uses control flow (read from top to bottom)
  • Uses conditional statements, for loops, and others to manipulate data
