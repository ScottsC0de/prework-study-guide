HTML


  • Stands for HyperText Markup Language
  • The structural langauge of a webpage
  • Composed of elements, tags and attributes to create a webpage's content
  • Responsible for what a user sees on a webpage
  • Also gives browser other instructions called metadata
  • Used with CSS and JavaScript to create a completed website

CSS


  • Stands for Cascading Style Sheets
  • The styling language of a webpage
  • Tells browser how to display HTML code
  • Composed of selectors, properties, and values
  • Provides a webpage's content with organization, positioning, aesthetics, and style

Git


  • A history of recorded changes to your code
  • Used for version control and debugging
  • Also allows for code collaboration between developers
  • Popurlarly used on a website called "GitHub.com"
  • Stored in software reposotories
  • Can be accessed through your computer's terminal, using git commands
  • A few examples of git commands:
    • git status: checks what branch you are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
    • git add -A: saves changes to your code

JavaScript


  • A dynamic, high-level, object-oriented programming language
  • Works with HTML and CSS to add interactivity to a webpage
  • Composed of variables, arrays, functions, if statements, and for loops, among other things, to power a static webpage
